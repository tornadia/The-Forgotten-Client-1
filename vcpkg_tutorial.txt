Download and install vulkan sdk installer for windows from https://vulkan.lunarg.com/sdk/home
no need to enable any optional install.

installing vcpkg:
using any command line program like cmd, do:
cd c:
git clone https://github.com/Microsoft/vcpkg
cd vcpkg
.\bootstrap-vcpkg.bat
.\vcpkg integrate install

installing required libs:
x64 libs:
./vcpkg install --triplet x64-windows sdl2[*] curl zlib

x86 libs:
./vcpkg install sdl2[*] curl zlib

the [*] in sdl2 name make sdl2 have vulkan support
